## Cara Convert ke PDF

### Opsi 1: Dari Browser (Recommended)
1. Buka `PRESENTATION_SLIDES.html` di browser
2. Print (Cmd+P / Ctrl+P)
3. Pilih "Save as PDF"
4. Settings: Landscape, Margin: Minimal

### Opsi 2: Online Converter
- Upload HTML ke online converter (HTML to PDF)
- Atau gunakan tools seperti: https://www.ilovepdf.com/html-to-pdf

### Opsi 3: Install PDF Engine
```bash
# Install pdflatex (untuk pandoc)
brew install basictex
# atau
brew install mactex

# Kemudian convert
pandoc docs/PRESENTATION_SLIDES.md -o docs/PRESENTATION_SLIDES.pdf \
  --pdf-engine=pdflatex \
  --css=docs/presentation-pdf-style.css \
  --standalone
```
